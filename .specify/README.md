# .specify

This folder contains Spec-Kit / Specify metadata for Project Chimera.

Purpose:
- Hold generated specs, templates, and configuration created by `specify` or other spec tooling.
- Provide a stable location for agents to read/write project specification artifacts.

Notes:
- If `specify` is available, run `specify init --here` to populate this directory with standard files.
- This README is intentionally minimal; keep machine-readable outputs in the same folder.
